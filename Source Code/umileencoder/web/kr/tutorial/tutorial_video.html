인코딩하기
인코딩 하기
유마일 인코더를 통해 동영상 변환 작업을 수행하는 방법은 다음과 같습니다.
변환할 동영상 추가
파일추가를 클릭하여 변환 작업 리스트에 파일을 추가합니다. 드래그 & 드롭을 통해 파일 추가 또한 가능합니다.
프리셋 선택
자신이 원하는 기기에 맞는 프리셋을 선택합니다. 만약, 프리셋을 변경하기 위해서는 아래 화살표를 클릭하면 자신이 원하는 기기를 선택할 수 있도록 프리셋 즐겨찾기 목록이 나타납니다.

프리셋 즐겨찾기 목록은 즐겨찾기 관리를 통해 사용자가 직접 관리할 수 있습니다. 자세한 설명은 즐겨찾기 관리를 통해 확인할 수 있습니다.
변환 시작
프리셋 선택이 완료되면, 변환시작 버튼 클릭을 통해 변환 작업을 시작합니다. 작업이 시작되면, 변환 속도와 남은 시간 등이 표시되며취소 버튼 클릭을 통해 변환 작업을 취소할 수 있습니다.

변환 작업 종료 후 컴퓨터 전원을 자동으로 종료하고 싶으면, 아래 스위치를 ON 으로 변경해두면 됩니다.
