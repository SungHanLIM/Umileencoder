프로그램 설치 시작
다운로드 받은 유마일 인코더 설치파일을 실행시킵니다.
사용자 계약 동의
유마일 인코더를 사용하기 위해서는 본 계약에 동의하셔야 합니다.
설치할 위치 선택
유마일 인코더가 설치될 폴더를 선택합니다. 특별한 경우가 아니면 별도의 설정없이 다음 버튼을 클릭합니다.
시작 메뉴 폴더 선택
유마일 인코더의 시작 메뉴 폴더를 선택합니다. 특별한 경우가 아니면 별도의 설정 없이 다음 버튼을 클릭합니다.
추가 사항 적용
유마일 인코더 설치 과정에서 추가로 적용하는 사항을 선택합니다. 바탕화면에 아이콘 설치를 원하지 않는 경우 체크 박스를 해제한 후 다음 버튼을 클릭합니다.
설치 준비 완료
유마일 인코더의 시작 매뉴 폴더를 선택합니다. 특별한 경우가 아니면 별도의 설정없이 다음 버튼을 클릭합니다.
설치 완료
유마일 인코더가 설치되는 진행과정이 나오고 이후 설치 완료 화면이 나타납니다. 완료 버튼을 클릭하면 설치가 완료됩니다.
