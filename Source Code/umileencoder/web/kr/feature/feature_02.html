GPU 가속기능을 활용한 동영상 변환 속도 향상!
유마일인코더는 동영상 변환 속도의 향상을 위해 NVIDIA CUDA 기술을 탑재하였습니다.
NVIDIA CUDA 기술을 적용하게 되면, 컴퓨터의 CPU 와 NVIDIA의 GPU를 함께 활용하여 동영상을 변환하기 때문에, 일반 CPU만을 활용하는 컴퓨터보다 훨씬 더 빠른 동영상 변환이 가능하게 됩니다.
아래 그림은 동영상 변환 시 1) 유마일인코더 (CPU only) , 2) 유마일인코더 with CUDA 경우의 변환 속도의 차이를 보여줍니다.
