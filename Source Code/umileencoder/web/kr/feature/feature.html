System Requirements
지원 운영체제 32bit or 64 bit editions of Windows XP (SP2 or higher), Windows Vista, and Windows 7
프로세서 Intel Pentium 4 또는 AMD 866MHz 이상
메모리 256MB 이상 (권장사양 : 512M 이상)
HDD 30MB 이상의 여유 공간 (권장사양 : 1GB 이상)
Etc Windows Media Player 9 이상 설치 권장
주의사항
  • - 비디오(영상)가 포함되지 않은 파일은 변환되지 않습니다.
  • - 원본 파일이 손상된 경우 변환되지 않을 수 있습니다.
  • - Window Media (WMV, ASF) 파일 변환의 경우, Windows Media Player 9 이상 설치를 권장합니다.
  • - 원본 파일의 평균 프레임레이트가 10fps 이하인 동영상은 , 정상 재생되지 않을 수 있습니다.
