マイプロファイル生成
Add user Defined Profile
Umile Encoderは多彩なディバイスのプロファイルを提供するだけではなく、ユーザの希望する設定値を入力することで、マイプロファイルが生成できる機能も提供します。
動画ファイルフォーマット設定
お客様の希望する動画のファイルフォーマットを選択します。
ビデオ&オーディオ設定変更
プロファイル生成のためにお客様の希望するビデオ、およびオーディオの設定値を変更します。
新しいプロファイルの保存
設定値変更が完了すると、新しいプロファイル生成のために「名前を付けて保存」をクリックします。すると、プロファイル名の入力を要求するポップアップウィンドウが出るので、新しい名前を入力してから「保存」をクリックして保存します。
