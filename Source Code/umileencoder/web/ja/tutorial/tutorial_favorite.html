お気に入り管理
Setup Favorite List
「お気に入り管理」はUmile Encoderから提供される多彩なディバイスのプロファイルをより楽に管理できるようにする機能で、希望するディバイスのプロファイルを選択してプロファイルお気に入りリストにして管理できます。
お気に入りリスト設定
プログラムをインストールした後、初めて実行する場合は、下のリストに登録されているプロファイルが存在しないので、プロファイルの選択作業を行う必要があります。
プロファイル選択
「お気に入り管理」をクリックすると、Umile Encoderから提供される多彩なディバイスのリストが出ます。お客様の希望するディバイスを選択した後、「OK」をクリックすると、「プロファイルお気に入りリスト」に追加されます (ディバイスは複数選択可能)。
お気に入りリストに登録完了
「お気に入り管理」から、選択したディバイスがリスト上に登録されていることを確認できます。この画面で希望するプロファイルを選択した後、動画変換作業を行います。ユーザはいつでもお気に入りリストにプロファイルを追加・削除できます。
