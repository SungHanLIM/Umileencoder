GPU 加速機能を 活用した動画の変換速度を向上.
動画の変換速度の向上のためにNVIDIA CUDAテクノロジーを搭載しています。
NVIDIA CUDA技術を適用すると、コンピュータのCPUとNVIDIAのGPUを一緒に活用して動画を変換するため、通常のCPUのみを活用しているコンピュータよりもはるかに高速に動画変換が可能になります。
下の図は、動画の変換時に1）UMILEエンコーダー（CPU only）、2）UMILEエンコーダおよびCUDAの場合の変換速度の違いを示しています。
