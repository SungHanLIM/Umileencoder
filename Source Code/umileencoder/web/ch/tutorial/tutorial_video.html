Video Conversion
使用 Umile Encoder 转换媒体格式按如下程序进行。
添加需要转换格式的视频
点击添加文件，以向转换目录上添加文件。也可通过拖放操作添加文件。
选择Profile
选择适合自己设备的Prifile。如果想变更预设，点击下面的箭头即出现预设收藏目录，可选择自己希望的设备。

预设收藏目录，可由用户直接管理。详细说明可通过收藏管理确认。
开始转换格式
完成预设选择后，即可点击开始转换按钮进行转换操作。开始操作后屏幕显示转换速度和剩余时间，按取消按钮即可取消转换操作。
 

如果想在完成转换操作后自动关掉电脑电源，可将下面的开关置ON。
