安装 Umile Encoder
Umile Encoder Installation
开始安装
运行下载完的 Umile Encoder 安装文件。
选择同意用户协议
要使用 Umile Encoder，需要同意该协议。
选择将要安装的位置
选择将安装 Umile Encoder 的目录。如果不需要特别设定，按下一步。
选择开始菜单目录
选择 Umile Encoder 的开始菜单目录。 如果不需要特别设定，按下一步。
其他设定
在安装 Umile Encoder 的过程中选择其他设定。如果不想在桌面生成快捷图标，取消复选框勾选。
准备安装
选择 Umile Encoder的开始菜单目录。 如果不需要特别设定，按下一步。
安装完成
显示 Umile Encoder 安装进行过程后显示安装完成屏幕。点击完成按钮完成安装。
