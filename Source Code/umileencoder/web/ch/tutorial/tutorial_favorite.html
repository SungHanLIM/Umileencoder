预设收藏管理
Setup Favorite List
预设收藏管理是由Umile Encoder提供的、可以更方便地管理多种设备预设的一种功能， 可以选择想使用的设备预设，通过预设收藏目录进行管理。
设定预设收藏目录
当安装程序后首次运行时，在下面的目录中没有预设内容，因此要进行选择预设的操作。
选择预设
点击预设管理即出现由Umile Encoder提供的多种设备列表。选择自己需要的设备后，点击OK键即可添加到预设目录。(可选多个设备)。
完成预设目录注册
可以通过目录确认通过预设管理选择的设备已注册。在这里选择需要的预设后即可进行视频格式转换。用户可随时向预设收藏目录添加预设或删除预设。
